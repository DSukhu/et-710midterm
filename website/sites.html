Question and Answer

Q:What is a Responsive Website?

A: Responsive web design allows a site’s layout to change as the screen size being used to view that site changes. A wide screen display can receive a site design with multiple columns of content while a small screen can have that same content presented in a single column with text and links that are appropriately sized to be read and used on that smaller display. It is important because in the long run, money and time will be saved. Also, content publishers have the option to build versions of their website for every popular device platform.

Q:What is Agile?

A:Aglie is a group of software development methodologies based on iterative development, where requirements and solutions evolve through collaboration between self-organizing cross-functional teams.

Q:What is Six Sigma?

A:Six Sigma is a method that provides organizations tools to improve the capability of business processes. The Six Sigma DMADV process (define, measure, analyze, design, verify) is an improvement system used to develop new processes or products at Six Sigma quality levels. It can also be employed if a current process requires more than just incremental improvement.
